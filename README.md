
# 🎨 Ai-to-pptx
Ai-to-pptx是使用AI技术来自动生成PPTX，并且支持在线修改和导出PPTX。
主要功能:
1 使用ChatGPT等大语言模型来生成大纲
2 生成的内容允许用户再次修改
3 生成PPTX的时候可以选择不同的模板
4 支持在线修改PPTX的文字内容，样式，图片等
5 支持导出PPTX，PDF，PNG等多种格式

# 🚀 前端项目运行
```
npm install

npm run dev
```

# 🚀 后端项目运行(另外打开一个终端)
```
cd express

npm install

npm run express
```
